## 🌱 O que é o BiotechProject?

BiotechProject é um **laboratório digital aberto** dedicado à **biotecnologia, saúde e divulgação científica** na web.

O projeto combina:
- 🧪 Conteúdo científico sobre biologia, medicina e biotecnologia
- 💻 Desenvolvimento web moderno (HTML, CSS, JavaScript)
- 📊 Monitoramento automático de desempenho com Lighthouse
- 🚀 Automação contínua com GitHub Actions

É um espaço colaborativo para desenvolvedores, pesquisadores e entusiastas que querem explorar **como a tecnologia pode servir à ciência**.

## 🚀 Principais recursos

✅ **Painel de desempenho automatizado**  
→ Análise contínua de todas as páginas com atualização de `performance-data.json`

✅ **Filtro dinâmico por categoria**  
→ Interface interativa para explorar o nível de maturidade tecnológica

✅ **Fluxo de trabalho CI/CD integrado**  
→ Testes, análises e atualizações automáticas a cada alteração

✅ **Design responsivo e acessível**  
→ Funciona em todos os dispositivos, com foco em usabilidade

✅ **Aberto por design**  
→ Aberto a contribuições, ideias e colaborações

---

## 🛠️ Tecnologias utilizadas

- **Frontend**: HTML5, CSS3, JavaScript (ES Modules)
- **Análise**: Lighthouse, Chrome Launcher
- **Automação**: GitHub Actions
- **Monitoramento**: `performance-data.json` + painel dinâmico
- **Implantação**: GitHub Pages

---

## 📊 Painel de Desempenho

O projeto inclui um sistema automatizado para monitorar a qualidade técnica das páginas:

- Pontuação de desempenho (0–100)
- Tempo de carregamento
- Categoria de maturidade: `optimized`, `compatible`, `needs-improvement`, `deprecated`
- Atualização diária via fluxos de trabalho

👉 Veja o relatório em tempo real: [Tech_Maturity.html](https://gitechnolo.github.io/biotechproject/Tech_Maturity.html)
