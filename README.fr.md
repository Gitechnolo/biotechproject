## 🌱 Qu'est-ce que BiotechProject ?

BiotechProject est un **laboratoire numérique ouvert** dédié à la **biotechnologie, à la santé et à la vulgarisation scientifique** via le web.

Le projet combine :
- 🧪 Contenu scientifique sur la biologie, la médecine et les biotechnologies
- 💻 Développement web moderne (HTML, CSS, JavaScript)
- 📊 Surveillance automatisée des performances avec Lighthouse
- 🚀 Automatisation continue via GitHub Actions

C'est un espace collaboratif pour développeurs, chercheurs et passionnés qui souhaitent explorer **comment la technologie peut servir la science**.

## 🚀 Fonctionnalités principales

✅ **Tableau de bord de performance automatisé**  
→ Analyse continue de toutes les pages avec mise à jour de `performance-data.json`

✅ **Filtre dynamique par catégorie**  
→ Interface interactive pour explorer le niveau de maturité technologique

✅ **Flux CI/CD intégré**  
→ Tests, analyses et mises à jour automatiques à chaque modification

✅ **Design responsive et accessible**  
→ Fonctionne sur tous les appareils, avec une attention portée à l'ergonomie

✅ **Ouvert par conception**  
→ Ouvert aux contributions, idées et collaborations

---

## 🛠️ Technologies utilisées

- **Frontend** : HTML5, CSS3, JavaScript (ES Modules)
- **Analyse** : Lighthouse, Chrome Launcher
- **Automatisation** : GitHub Actions
- **Surveillance** : `performance-data.json` + tableau de bord dynamique
- **Déploiement** : GitHub Pages

---

## 📊 Tableau de bord des performances

Le projet inclut un système automatisé pour surveiller la qualité technique des pages :

- Score de performance (0–100)
- Temps de chargement
- Catégorie de maturité : `optimized`, `compatible`, `needs-improvement`, `deprecated`
- Mise à jour quotidienne via les workflows

👉 Voir le rapport en temps réel : [Tech_Maturity.html](https://gitechnolo.github.io/biotechproject/Tech_Maturity.html)
