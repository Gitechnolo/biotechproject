## 🌱 ¿Qué es BiotechProject?

BiotechProject es un **laboratorio digital abierto** dedicado a la **biotecnología, la salud y la divulgación científica** a través de la web.

El proyecto combina:
- 🧪 Contenido científico sobre biología, medicina y biotecnología
- 💻 Desarrollo web moderno (HTML, CSS, JavaScript)
- 📊 Monitoreo automático de rendimiento con Lighthouse
- 🚀 Automatización continua mediante GitHub Actions

Es un espacio colaborativo para desarrolladores, investigadores y entusiastas que quieran explorar **cómo la tecnología puede servir a la ciencia**.

---

## 🚀 Características principales

✅ **Panel de rendimiento automatizado**  
→ Análisis continuo de todas las páginas con actualización de `performance-data.json`

✅ **Filtro dinámico por categoría**  
→ Interfaz interactiva para explorar el estado de madurez tecnológica

✅ **Flujo de trabajo CI/CD integrado**  
→ Pruebas, análisis y actualizaciones automáticas en cada cambio

✅ **Diseño responsive y accesible**  
→ Funciona en todos los dispositivos, con enfoque en usabilidad

✅ **Abierto por diseño**  
→ Abierto a contribuciones, ideas y colaboraciones

---

## 🛠️ Tecnologías utilizadas

- **Frontend**: HTML5, CSS3, JavaScript (ES Modules)
- **Análisis**: Lighthouse, Chrome Launcher
- **Automatización**: GitHub Actions
- **Monitoreo**: `performance-data.json` + panel dinámico
- **Despliegue**: GitHub Pages

---

## 📊 Panel de Rendimiento

El proyecto incluye un sistema automático para monitorear la calidad técnica de las páginas:

- Puntuación de rendimiento (0–100)
- Tiempo de carga
- Categoría de madurez: `optimized`, `compatible`, `needs-improvement`, `deprecated`
- Actualización diaria mediante flujos de trabajo

👉 Ver informe en tiempo real: [Tech_Maturity.html](https://gitechnolo.github.io/biotechproject/Tech_Maturity.html)
