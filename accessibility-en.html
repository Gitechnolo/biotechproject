📌 Accessibility Statement

BiotechProject is committed to ensuring that the project is accessible to everyone, including users with visual, auditory, cognitive, or motor disabilities.

🎯 Compliance with Standards

This website follows the principles of the Web Content Accessibility Guidelines (WCAG) 2.1, level AA.

✅ Accessibility Features

  • Keyboard navigation: All functionality is accessible using only the Tab, Enter, Space, and Esc keys.
  • Screen reader compatibility: Semantic HTML structure, ARIA labels, and dynamic states are properly managed.
  • Text/background contrast: All text meets the minimum 4.5:1 contrast ratio required by WCAG AA.
  • Adaptable text: Content remains readable even at 200% zoom.
  • Animations: Visual effects are optional; disabled if the user prefers prefers-reduced-motion.

🔄 Update: Accessible Dropdown Menu (2025)

We've improved the accessibility of the main menu by introducing:

  • Use of aria-label to clearly describe the function of each menu item
  • aria-controls attribute to explicitly link buttons to their dropdowns
  • Replacement of <a href="javascript:..."> with <button> for correct semantic structure
  • Full keyboard navigation support (Tab, Enter, Esc)
  • Maintained backward compatibility with existing documents

These changes enhance the experience for users with visual and motor disabilities, ensuring fairer access to content.

🎨 New: Dynamic Visual Theme (2025)

We've introduced a color theme switcher directly in the Service menu to offer a personalized and more accessible visual experience.

  • Customizable colors: choose between Green, Cyan, Violet, Orange, and Deep Blue to adapt the interface to your visual preferences.
  • Colorblind support: some colors (e.g. Deep Blue) are easier to distinguish for users with certain color vision deficiencies.
  • Automatic adaptation: if the system detects prefers-contrast: high, the default theme becomes Deep Blue for better readability.
  • Saved preference: your selected theme is stored in the browser and restored on your next visit.
  • Keyboard and screen reader accessible: the button is navigable with Tab and the label updates dynamically to describe the current theme.

This tool is particularly useful for:

  • Users with light sensitivity or visual contrast issues
  • Those using the site in very bright or very dark environments
  • Users with color blindness or other visual conditions

The dynamic theme is an example of how futuristic design and accessibility can coexist — without compromise.

🛠️ Technologies Used

HTML5, CSS3, JavaScript (vanilla), ARIA, WAI-ARIA.
